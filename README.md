# Portfolio

A modern, animated portfolio built with React, Vite, Tailwind CSS, and Framer Motion.

## Features

- Smooth scroll animations
- Custom cursor
- Dark/Light theme toggle
- Responsive design
- Magnetic button effects
- Animated counters

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Framer Motion
- Lenis (smooth scroll)
